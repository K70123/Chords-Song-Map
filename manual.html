    Right-click the songs you want to remove from the live section to remove it

  • Main Song Display

    • Key Buttons

    Double-click one of the key buttons (except num button) to set the orignial key of the song

    Click on the key that you want the chords to transpose to

    Edit Chords

    Change the amount of beats per bar by double-clicking the number next to the 'beats per bar: ' span

    Double-click the chords to edit it, you can move to the next chord bar by pressing 'Enter'

    If the bar don't have any beats or chords in it, it automatically deletes the entire bar

    Edit Chord Section

    Double-click the title of each section (i.e intro) to rename the chord section

    Click the + button at the bottom of each chord section to add a new chord section

    Song Map

    Click the + button to add new song map part

    Double-click the song map part you want to rename it

    If the song map part is empty it will automatically delete it
